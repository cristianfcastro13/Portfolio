This website serves as my personal portfolio, designed to showcase my skills, projects, and professional experience. Built from the ground up with HTML, CSS, and JavaScript, it provides a central hub for potential employers and collaborators to learn more about my capabilities and see my work in action.






The challenge was to create a compelling and user-friendly online presence that effectively communicates my technical skills and design sensibilities. This involved designing an intuitive user interface, developing a responsive layout for various devices, and ensuring optimal performance and maintainability.

As the sole creator of this portfolio website, I was responsible for all aspects of its development, including:

  • Conceptualization and Design: Planning the website's structure, user flow, and visual design (wireframing, mockups).
  • Front-End Development: Building the user interface using HTML, CSS, and JavaScript.
  • Back-End Development: Implementing form integration for the contact section in the main page.
  • Deployment and Hosting: Setting up the website on the hosting platform 'GitHub Pages'
  • Content Creation: Writing all the text and preparing the visuals for the site.
